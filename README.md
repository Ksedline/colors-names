Этот репозиторий - огромный список цветов и сопоставленных им HEX и RGB кодов.
Украдено отсюда: https://colorscheme.ru/color-names.html

# Схема JSON
```golang
type color struct {
    Name string // Название цвета
    Hex  string // HEX Код

    R    uint16 // RGB - Red
    G    uint16 // RGB - Green
    B    uint16 // RGB - Blue
}
```
